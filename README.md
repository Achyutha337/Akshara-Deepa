## 🌟 About

**Akshara-Deepa** (ಅಕ್ಷರ ದೀಪ — *Light of Letters*) is an intelligent self-study companion designed specifically for 10th-grade SSLC students. It empowers learners to take charge of their own education through structured progress tracking, personalized quizzes, and smart performance analytics — all in a student-friendly interface.

---

## ✨ Features

| Feature | Description |
|---|---|
| 📖 **Chapter-wise Progress Tracking** | Monitor your study progress for every chapter across all subjects |
| 🧠 **Personalized Quizzes** | AI-generated quizzes tailored to your learning gaps |
| 📊 **Performance Analysis** | Visual insights into strengths and weak areas |
| ⚠️ **Weak Subject Identification** | Automatically flags subjects and topics needing more attention |
| 🎯 **Guided Self-Learning** | Structured learning paths to maximize exam preparation |
| 🖥️ **Student-Friendly UI** | Clean, interactive interface designed for young learners |

## 🚀 Getting Started

### Prerequisites

- Android Studio (latest stable version)
- JDK 11 or higher
- Android SDK (API level 24+)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Achyutha337/Akshara-Deepa.git
   cd Akshara-Deepa
   ```

2. **Open in Android Studio**
   - Launch Android Studio
   - Select **Open an existing project**
   - Navigate to the cloned folder and open it

3. **Build the project**
   ```bash
   ./gradlew build
   ```

4. **Run on a device or emulator**
   - Connect an Android device or start an emulator
   - Click **Run ▶** in Android Studio
